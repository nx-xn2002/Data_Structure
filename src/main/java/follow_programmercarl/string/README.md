# 字符串操作

## 反转字符串

[344. 反转字符串](https://leetcode.cn/problems/reverse-string/)

由于要求原地反转，直接使用双指针标记头尾开始遍历，并交换相应位置的值即可

- 时间复杂度O(n)
- 空间复杂度O(1)

## 反转字符串 II

[541. 反转字符串 II](https://leetcode.cn/problems/reverse-string-ii/)

即每隔k位进行一次反转，注意读题

- 时间复杂度O(n)
- 空间复杂度O(1)
